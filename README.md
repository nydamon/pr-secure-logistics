- **NDAA-Compliant Security**: Surveillance cameras, access control, and security hardware from authorized partners (Axis, Hanwha)
- **NSA-Listed Shredders (P-7)**: TAA-compliant shredders for secure document destruction
- **Ruggedized Laptops (MIL-STD)**: MIL-STD-810G certified devices from Panasonic (Toughbook) and Dell
- **TAA-Compliant IT Infrastructure**: UPS systems, server racks, network switches from Eaton, APC, and Cisco
- **Secure Server Racks & SCIFs**: RF-shielded cabinets and SCIF-compliant hardware
- **Authorized Supply Chain**: Products sourced directly from Tier-1 distributors (TD Synnex, Ingram Micro)

### Deployment

This site is designed to be deployed on static hosting platforms:

#### Vercel (Recommended)
1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect and deploy the static site
3. Custom domain can be configured in Vercel settings

#### Other Options
- **Netlify**: Drag and drop the folder or connect via Git
- **GitHub Pages**: Enable in repository settings
- **AWS S3 + CloudFront**: For enterprise hosting
- **Any static hosting service**: The site is a single HTML file with assets

## ğŸ”§ Customization

### Updating Content

Edit `index.html` directly. The site uses Tailwind CSS utility classes, making it easy to modify:
- Colors: Change `bg-blue-600` to your preferred color
- Spacing: Adjust padding/margin classes (`py-24`, `px-6`, etc.)
- Typography: Modify font sizes (`text-4xl`, `text-lg`, etc.)

### Adding Images

1. Place images in `assets/images/`
2. Update image paths in `index.html`
3. Use optimized/cropped versions for better performance

### Changing Logo

- Header logo: `PR secure logistics_cropped.png`
- Footer logo: `transparant_transparent.png`
